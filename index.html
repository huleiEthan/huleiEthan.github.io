添加samba账户

这是因为没有加相应的系统账号，所以会提示Failed to add entry for user的错误，只需增加相应的系统账号samba就可以了:
[root@ubuntu ~]# groupadd share -g 6000
[root@ubuntu ~]# useradd samba -u 6000 -g 6000 -s /sbin/nologin -d /dev/null

这时就可以用smbpasswd -a samba增加samba这个samba账号了!为了增加系统的安全性，所以加的系统账号不要给shell它，也不给它指定目录，到时在/home目录给samba账号建个文件夹，该文件夹只有samba有读写权限即可!
如:
[root@ubuntu ~]# mkdir /home/samba
[root@ubuntu ~]# chown -R samba:share /home/samba

若不想让另人访问，只让samba用户可以访问，只需执行命令:
[root@ubuntu ~]# chmod u+rwx,g+rwx,o-rwx /home/samba

重启服务

windows客户端连接到samba服务器

有两种方法：
1、网上邻居–>查看工作组计算机–>Samba Server
2、打开运行（windows键+R），输入配置samba服务器的主机名或者IP（格式： \192.168.1.98）
